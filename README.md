## 项目介绍
本项目是基于PXA270的开发板实现的一个电子相册，但由于项目的扩展功能，使得本项目可以用于“限时猜国旗”游戏。以下是基本操作的介绍。    
![PXA270EAlbum](Manuals/EALBUM.jpg)

* 开机时，数码管显示时间
* 按键1，显示当前日期
* 按键3，成功显示文字，再次按下关闭文字显示
* 按键5，LCD显示上一张图片
* 按键6，LCD显示下一张图片
* 按键7，启动图片轮播，LCD上的照片进行循环播放，再次按下即停止轮播
* 按键9，重置时间，日期以及图片

更多详尽内容，请参考Manuals中的*设计报告*。
